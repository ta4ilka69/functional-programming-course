## Цель работы

Основной целью лабораторной работы является освоение работы с:
- Вводом/выводом в Haskell
- Потоковой обработкой данных
- Аргументами командной строки
- Интерполяцией данных

В рамках работы реализована программа для интерполяции точек с использованием линейного метода и метода Лагранжа.

## Требования к разработанному ПО

### Реализованный интерфейс:
- **Парсинг аргументов командной строки**: выбор алгоритма и шага интерполяции
- **Потоковая обработка входных данных**: чтение точек из стандартного ввода
- **Методы интерполяции**: 
  - Линейная интерполяция (по двум точкам)
  - Интерполяция методом Лагранжа (по четырём точкам)
- **Форматированный вывод**: результаты в табличном виде

### Дополнительные требования:
- Программа должна корректно обрабатывать ошибки ввода
- Поддержка различных режимов работы (linear, lagrange, both)
- Возможность задания шага интерполяции

## Ключевые элементы реализации

### Модуль [CommandLineParser](./CommandLineParser.hs)

Модуль отвечает за парсинг аргументов командной строки:


Основные компоненты:
- **Settings**: тип данных для хранения настроек программы
- **parseArgs**: функция разбора аргументов командной строки
- **parseAlgorithm**: преобразование строкового представления алгоритма в тип Algorithm

### Модуль [Interpolation](./Interpolation.hs)

Модуль содержит реализацию алгоритмов интерполяции:


Ключевые функции:
- **linearInterpolation**: реализация линейной интерполяции между двумя точками
- **lagrangeInterpolation**: реализация интерполяции методом Лагранжа
- **Algorithm**: тип данных для выбора метода интерполяции

### Модуль [StreamProcessing](./StreamProcessing.hs)

Модуль обеспечивает потоковую обработку входных данных:


Основные функции:
- **processStream**: организация потоковой обработки входных данных
- **parsePoint**: парсинг строки в точку (x, y)
- **processPoints**: обработка набора точек с применением выбранного алгоритма

### [Main](./main.hs)

Главный модуль программы, координирующий работу остальных компонентов:


Основные функции:
- **main**: точка входа в программу, обработка аргументов
- **loop**: основной цикл обработки входных данных
- **printInterpolation**: форматированный вывод результатов

## Особенности реализации

1. **Потоковая обработка**:
   - Построчное чтение входных данных
   - Немедленный вывод результатов

2. **Алгоритмы интерполяции**:
   - Линейная интерполяция использует две последние точки
   - Метод Лагранжа использует четыре последние точки
   - Возможность комбинирования методов
